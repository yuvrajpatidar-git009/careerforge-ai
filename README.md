# 🚀 CareerForge AI

> **Build Your Career. Build Your Brand.**

CareerForge AI is an AI-powered career development platform that helps students, developers, freelancers, and job seekers create professional career assets from a single dashboard.

Instead of using multiple websites for resumes, portfolios, LinkedIn optimization, GitHub README generation, and cover letters, CareerForge AI provides everything in one modern platform.

# 🎯 Problem Statement

Students and professionals often use multiple platforms to create career documents such as resumes, portfolios, LinkedIn profiles, GitHub READMEs, and cover letters.

This process is time-consuming and repetitive.

CareerForge AI solves this problem by providing all essential career-building tools in one platform.

---

# 💡 Solution

CareerForge AI combines multiple AI-powered tools into a single application that allows users to create, manage, and improve their professional identity efficiently.

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/your-username/careerforge-ai.git
```

## Navigate to the Project

```bash
cd careerforge-ai
```

## Install Dependencies

```bash
npm install
```

## Configure Environment Variables

Create a `.env.local` file and add the required environment variables.

Example:

```env
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=

DATABASE_URL=

NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=

GEMINI_API_KEY=

STRIPE_SECRET_KEY=
```

## Run Development Server

```bash
npm run dev
```

Visit:

```
http://localhost:3000
```
