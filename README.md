CareerForge AI
Tagline: Build Your Career. Build Your Brand.
CareerForge AI is an AI-powered career development platform designed to help students, developers, freelancers, and job seekers build their professional identity from a single dashboard. Instead of using multiple websites for resumes, portfolios, LinkedIn optimization, GitHub README generation, and cover letters, CareerForge AI brings everything together in one modern and easy-to-use platform.
________________________________________
Project Overview
The goal of CareerForge AI is to simplify the career-building process by providing AI-powered tools that generate professional career assets quickly and efficiently. Users can create resumes, portfolio websites, LinkedIn content, GitHub READMEs, and cover letters while managing everything from one centralized dashboard.

Problem Statement
Students and professionals often rely on multiple platforms to create resumes, portfolios, LinkedIn profiles, GitHub READMEs, and cover letters. This process is repetitive, time-consuming, and inefficient.
CareerForge AI solves this problem by providing all essential career-building tools within a single AI-powered platform.
________________________________________
Solution
CareerForge AI combines multiple AI-powered services into one application, enabling users to generate, organize, edit, and manage all their professional career assets from one dashboard.

Getting Started
Clone the repository:
git clone https://github.com/your-username/careerforge-ai.git
Navigate to the project directory:
cd careerforge-ai
Install all dependencies:
npm install
Create a .env.local file and configure the required environment variables.
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
DATABASE_URL=
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
GEMINI_API_KEY=
STRIPE_SECRET_KEY=
Run the development server:
npm run dev
Open your browser and visit:
http://localhost:3000
